# 部署到Vercel - 实现公网访问

## 📋 部署步骤

### 方法一：通过Vercel网站部署（推荐）

1. **访问Vercel网站**
   - 打开 https://vercel.com
   - 使用GitHub账号登录（如果没有账号，先注册）

2. **导入项目**
   - 点击 "Add New Project"
   - 选择你的GitHub仓库（如果还没有，需要先上传到GitHub）
   - 或者点击 "Import Git Repository" 并选择你的仓库

3. **配置项目**
   - Framework Preset: 选择 "Other" 或 "Static Site"
   - Root Directory: 留空（或填写 `.`）
   - Build Command: 留空（因为是静态文件）
   - Output Directory: 留空（或填写 `.`）

4. **部署**
   - 点击 "Deploy"
   - 等待部署完成（通常1-2分钟）

5. **获取公网地址**
   - 部署完成后，Vercel会给你一个地址，格式类似：
     `https://your-project-name.vercel.app`
   - 这个地址就是你的公网访问地址

### 方法二：通过Vercel CLI部署

如果你已经安装了Vercel CLI：

```bash
# 安装Vercel CLI（如果还没安装）
npm i -g vercel

# 在项目目录下登录
vercel login

# 部署
vercel

# 如果需要生产环境部署
vercel --prod
```

### 方法三：通过GitHub自动部署

1. **上传代码到GitHub**
   - 在GitHub创建新仓库
   - 上传所有文件到仓库

2. **连接Vercel**
   - 在Vercel中导入GitHub仓库
   - 设置自动部署（每次push代码自动部署）

## ✅ 部署后的效果

部署成功后：
- ✅ 二维码会自动使用公网地址（如 `https://your-project.vercel.app/index.html`）
- ✅ 任何人都可以通过微信扫描二维码访问签到页面
- ✅ 不需要在同一WiFi网络下
- ✅ 祝福数据会保存在浏览器的localStorage中（每个设备独立）

## 🔍 验证部署

1. 打开你的Vercel项目地址：`https://your-project.vercel.app/wall-with-qr.html`
2. 检查二维码是否显示
3. 用微信扫描二维码，应该能正常打开签到页面

## ⚠️ 注意事项

1. **localStorage限制**：由于使用localStorage，不同设备之间的数据不会同步
   - 如果需要多设备同步，需要后端API支持

2. **HTTPS**：Vercel自动提供HTTPS，微信扫码可以正常访问

3. **域名**：Vercel会提供一个免费域名，你也可以绑定自己的域名
