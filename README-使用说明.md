# 使用说明 - 无法打开网页的解决方案

## 问题排查步骤

### 1. 检查服务器是否运行

打开命令提示符或PowerShell，运行：
```bash
netstat -ano | findstr :3000
```

如果看到 `LISTENING` 状态，说明服务器正在运行。

### 2. 测试本地访问

在浏览器中打开：
```
http://localhost:3000
```

如果本地可以访问，说明服务器正常，问题可能是：
- 防火墙阻止了外部访问
- IP地址不正确
- 手机和电脑不在同一网络

### 3. 添加防火墙规则（需要管理员权限）

**方法一：使用命令（以管理员身份运行PowerShell）**
```powershell
netsh advfirewall firewall add rule name="Node.js Server 3000" dir=in action=allow protocol=TCP localport=3000
```

**方法二：手动设置**
1. 打开"Windows Defender 防火墙"
2. 点击"高级设置"
3. 点击"入站规则" → "新建规则"
4. 选择"端口" → "TCP" → 输入端口号 `3000`
5. 选择"允许连接"
6. 完成

### 4. 获取正确的IP地址

在命令提示符中运行：
```bash
ipconfig
```

查找"IPv4 地址"，通常是：
- `192.168.x.x`
- `10.x.x.x`
- `172.16.x.x` 到 `172.31.x.x`

### 5. 确保手机和电脑在同一WiFi

- 手机和电脑必须连接同一个WiFi网络
- 不能一个用WiFi，一个用手机热点（除非电脑连接手机热点）

### 6. 使用本地地址测试

如果局域网无法访问，可以：
1. 在电脑上打开二维码页面
2. 使用截图工具保存二维码
3. 通过微信/QQ发送到手机
4. 手机扫码访问

## 快速解决方案

### 方案A：使用本地地址 + 截图二维码

1. 在电脑浏览器打开：`http://localhost:3000/qrcode.html`
2. 截图保存二维码
3. 通过微信/QQ发送到手机
4. 手机扫码访问

### 方案B：使用手机热点

1. 打开手机热点
2. 电脑连接手机热点
3. 在电脑上运行服务器
4. 手机直接访问电脑的IP地址

### 方案C：使用ngrok等内网穿透工具

如果需要外网访问，可以使用：
- ngrok
- frp
- 其他内网穿透工具

## 测试连接

运行 `测试访问.bat` 或 `打开二维码页面.bat` 来快速测试。

2. 本地访问是否正常
3. 防火墙设置情况
4. IP地址信息
